You, the developer, are tasked with solving certain problems to gain stars that progressively ornament a Christmas tree. One problem is released every day. This repository contains work done in CSC385 in the Fall of 2023. The course covered searching, sorting, recursion, stacks, queues, arrays, deques, trees, hashing, and the study of algorithmic complexity and Big-Omega, Big-Theta, and Big-O notation.



This course involved an overview of the major components of many programming languages, including how to write if and switch-case statements, various loops, typing and binding, and subprograms, showing how these concepts are implemented in C, Java, C++, Fortran, Pascal, Ada, Cobol, Scheme, Prolog, Perl, Python, Ruby, and F#. There was one assignment drawing parse trees and working with grammars, and there were two required programming assignments: one in Java and one in Scheme. A third, bonus programming assignment was the conversion of the code from Scheme to Prolog, given some restrictions on allowed symbols, in one line.
